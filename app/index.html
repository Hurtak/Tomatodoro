What is this?

Application for time management inspired by technique called Pomodoro. The technique uses a timer to break down your working time into 25 minutes long work periods separated by short breaks. The method is based on the idea that frequent breaks can improve mental agility.

The most common implementation is as follows:

  1. 25 minutes of work
  2. 5 minute break
  3. Repeat four times, after which you have a 20–30 minute break
  4. Start again

For more detailed information see the very helpful Pomodoro get started guide.
